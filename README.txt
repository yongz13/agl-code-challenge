How to run the angular app
step 0: unzip the zip file and copy to your local drive
Step 1: Install NodeJs
Step 2: Install @Angular/Cli
Step 3: Install Visual Studio Code
step 4: open visual studio code
step 5: open the folder agl-code-challenge where the project file was copied to
step 6: Run npm install in vs code terminal
step 7: run ng serve in the terminal
step 8: type localhost:4200 in browser address bar then enter
step 9: enjoy the results. Cheers
