How to run angular app
step 0: unzip the zip file and copy to your local drive
Step 1: Install the latest NodeJs globally
step 2: open the folder agl-code-challenge in file explorer where the project was copied to
step3: open git bash from the opened folder in file explorer
step 6: Run npm install in git bash from the opend folder
step 7: run ng serve in git bash
step 8: type localhost:4200 in address bar abd enter
step 9: enjoy the results. Cheers
